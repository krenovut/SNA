# SNA
SNA project

Будем анализировать граф друзей в vk.com

Данные хранятся в следующем виде:  

'''
{
  "id_1": [
    id_3,
    id_4,
    id_5
  ],
  "id_2": [
    id_6,
  ],
  ...
  ]
}
'''
